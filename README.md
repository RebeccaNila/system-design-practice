# System Design Practice

This repository contains code examples and exercises for practicing System Design, specifically focusing on Low Level Design (LLD) and SOLID principles using Java.

## Project Structure

The code is organized by topic within the `src/main/java/dev/nila/practice` package.

### Covered Topics

- **SOLID Principles**
  - **Dependency Inversion Principle (DIP)**: Examples demonstrating violations and correct implementations of DIP.

## Getting Started

### Prerequisites

- Java Development Kit (JDK) 11 or higher
- Maven (optional, if using Maven for build)
- An IDE (IntelliJ IDEA, Eclipse, VS Code)
